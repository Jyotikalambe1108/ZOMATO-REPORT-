# **Zomato Analysis Dashboard**

## **Description**
This Power BI dashboard provides a detailed analysis of Zomato's restaurant data, offering insights into the total number of restaurants, cuisine diversity, average costs, ratings, and geographical distribution by country and city.

## **Key Performance Indicators (KPIs)**
* **Total Restaurants:** 7,433
* **Cuisine Count:** 146
* **Average Cost:** 1.20K
* **Average Rating:** 2.67

## **Total Restaurants by Country**
The map visualization shows the global presence and concentration of Zomato-listed restaurants, with the highest density appearing in the Asia/India region.

## **Total Restaurants by City**
The distribution of restaurants across key cities is as follows (ordered by count):
1.  **New Delhi**
2.  **Gurgaon**
3.  **Noida**
4.  **Faridabad**
5.  **Ghaziabad**
6.  **Albany**
7.  **Ahmedabad**

## **Total Restaurants by Rating Color**
This breakdown analyzes restaurant counts based on their assigned rating color, which typically correlates to a star rating range:
1.  **Orange** (Highest Count)
2.  **Yellow**
3.  **Red**
4.  **Green**
5.  **Dark Green** (Lowest Count)

## **Technologies Used**
* **Power BI Desktop**
* **DAX** for calculated columns and measures
* **Power Query** for data transformation
